Beschreibung
============================================
Das Modul time_conversion.py verarbeitet die von uns erhaltenen Daten aus der od.fmi-Datenbank. Die Berechnungen für die Semesterdaten, Vorlesungsstart- und Vorlesungsendzeiten, sowie die Daten für das aktuelle Semester werden dabei in seperaten Methoden berechnet.

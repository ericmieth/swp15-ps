Beschreibung
============================================
Das Modul ical.py erzeugt aus den erhaltenen Daten eine iCal-Struktur, welche für die iCal-Datei, welche als letzter Schritt downloadbar ist, notwendig ist. Hierfür nutzt sie zwei Funktionen, mk_ical und mk_fortnight, wobei die Erstere die Daten erzeugt und die zweitere aus diesen erzeugten Daten ein Zwei-Wochen-Kalender-Format erstellt.
